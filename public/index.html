PORTFOLIO

PAULO PORTFOLIO

This is my first portfolio, I did it with the purpose of learning and to show the projects I'm working on. I created with the idea of ​​being a static, simple and practical website, using JavaScript, HTML and CSS to learn web development.

JAVASCRIPT - HTML5 - CSS3

INVENTORY CONTROL

This is a simple inventory control project, using JavaScript, HTML5 and CSS3. I did it with the intuition of putting into practice what I am learning in the JS course and with that I put functions to add and remove items from a stock list, being able to remove the items in their entirety or item by item, and with verification functions entry and exit of information.

JAVASCRIPT - HTML5 - CSS3

WEB APP TAX SCHEDULE

Web application using Python, Django and JavaScript to obtain the tax agenda of the Federal Revenue and generate a text file in the csv model for download or an integration file for the Microsoft Outlook calendar depending on the user's choice.

PYTHON - DJANGO - JAVASCRIPT - JQUERY - HTML5 - CSS3

ICMS CALCULATOR

WebSite created to easily and quickly calculate the ICMS tax, showing the calculation memory used and a basic explanation of the ICMS, the website was created using JavaScript, HTML5 and CSS3, the Yarn package manager was also used, together with the Babel compiler, for learning purposes.

JAVASCRIPT - HTML5 - CSS3 - YARN - BABEL
